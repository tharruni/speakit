# SpeakIt вњЁ

Say what you really feel вЂ” SpeakIt finds the words.

An emotional intelligence web app. Pick how you feel, who you're talking to,
describe the situation, and get three perfectly toned messages (Soft, Direct,
Heartfelt) generated by Claude. Includes Translate Anger mode, Reverse Mode,
message history, and mood insights.

## Tech Stack

- **Frontend:** React, plain CSS (no framework)
- **Backend:** Node.js, Express
- **Database:** SQLite
- **Auth:** JWT + bcryptjs
- **AI:** Gemini API

## Setup вЂ” Backend

```bash
cd backend
npm install
cp .env.example .env
```

Open `.env` and add your real Gemini API key:

```
PORT=3001
JWT_SECRET=speakit_secret_2026_change_this
Gemini_API_KEY=your_claude_api_key_here
```


Run the backend:

```bash
npm start
```

You should see:

```
SpeakIt server running at http://localhost:3001/
```

This also auto-creates `speakit.db` with three tables: `users`, `messages`,
`moods`.

## Setup вЂ” Frontend

Open a **second terminal**:

```bash
cd frontend
npm install
cp .env.example .env
npm start
```

This opens `http://localhost:3000` in your browser. The frontend talks to the
backend at `http://localhost:3001` (set in `.env`).

## How To Use

1. Register a new account (name, email, password)
2. Login
3. **Generate tab:** pick an emotion, pick who you're talking to, describe
   what happened, hit Generate
4. **Anger Mode tab:** type raw unfiltered feelings, get a sendable version
5. **Reverse tab:** paste a confusing message you received, get it decoded
6. **History tab:** see and delete past messages
7. **Insights tab:** see your most-felt emotions and message stats

## API Routes (Backend)

| Method | Route | Auth | Description |
|---|---|---|---|
| POST | `/register` | No | Create account |
| POST | `/login` | No | Get JWT token |
| POST | `/generate` | Yes | Generate 3 toned messages |
| POST | `/translate-anger` | Yes | Transform raw message |
| POST | `/reverse-mode` | Yes | Decode a received message |
| GET | `/history` | Yes | Get last 50 messages |
| DELETE | `/history/:id` | Yes | Delete one message |
| GET | `/insights` | Yes | Mood stats for dashboard |
| GET | `/profile` | Yes | Logged-in user info |

All authenticated routes require header:
`Authorization: Bearer <jwt_token>`

- SpeakIt is not a replacement for therapy or real conversation вЂ” see the
